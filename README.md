# Trabajo Final UTN

## Descripción
Aplicación desarrollada en React que incluye sistema de registro con validaciones y persistencia de sesión mediante Context API; Esta misma simula un tipo de aplicaion de comunicacion, mediante mesnajes escritos.

## Tecnologías
- HTML
- CSS
- JavaScript
- React
- Vite
- React Router
- Context API
- LocalStorage

## Instalación

```bash
git clone https://github.com/mate-baldassini/trabajo-final-UTN.git
cd trabajo-final-UTN
npm install
npm run dev
